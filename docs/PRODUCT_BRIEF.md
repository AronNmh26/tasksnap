# Week 13 Product Brief (TaskSnap)

**Who It’s For**
Students and busy individuals who need a simple way to capture and manage tasks quickly.

**Problem It Solves**
People forget tasks because they are not captured immediately or reminders are not set.

**Core Value**
TaskSnap lets users create tasks in seconds and remember them with simple reminders.

**MVP Features**
1. Create tasks fast.
2. Edit and complete tasks.
3. Optional image attachment.
4. Reminders.

**User Feedback Plan**
1. Give the app to 3 classmates.
2. Ask them to add a task, edit it, and delete it.
3. Record what confused them or slowed them down.
4. Fix the top 3 issues.

**Success Signals**
1. Users can complete the main flow without help.
2. No crashes during a 5–10 minute test.
3. Users say the app is easy to understand.

**Next Improvements**
1. Reduce steps to add a task.
2. Improve reminder UX.
3. Clarify labels and empty states.
