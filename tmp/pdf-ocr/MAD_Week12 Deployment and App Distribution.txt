Action: Configure the Build
Your build tool (Gradle) needs to know where to find your key and password, but you must never
commit these secrets to version control. We use a .properties file that is ignored by Git.
The Safe Method:
1. Create a file named android/key.properties (this file should be in .gitignore).
2. Add your secret credentials to this file.
Al Prompt for Safe Configuration
Use this prompt to get help wiring it up without revealing your secrets.

file. Show me the code to add to android/app/build.gradle in a Flutter project to configure the
signingConfigs for the ‘release’ build type. Do not show any hardcoded passwords.”
